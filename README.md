# cringe.company — заглушка для домена

Это готовый статический сайт-заглушка для GitHub Pages:

- красивый full-screen landing с анимациями
- адаптивный дизайн (desktop + mobile)
- кастомный домен `cringe.company` через файл `CNAME`
- базовая SEO-конфигурация (`robots.txt`, `sitemap.xml`)
- кастомная `404` при ошибках перехода

## Структура репозитория

- `index.html` — основная страница
- `CNAME` — домен для GitHub Pages
- `robots.txt` — разрешения для индексации
- `sitemap.xml` — карта сайта
- `404.html` — страница ошибки

## Как запустить

1. Создай репозиторий `cringe-company` и запушь файлы.
2. В **Settings → Pages** включи публикацию с ветки `main` и папки `/ (root)`.
3. В DNS у регистратора добавь запись:
   - `CNAME` на `your-username.github.io`
4. Подожди распространение DNS (обычно до 24 часов).
5. В `CNAME` в репозитории остаётся `cringe.company`, это автоматически свяжет Pages с доменом.

## Быстрые изменения

- Меняй тексты/таймер/почту прямо в `index.html`.
- Для смены визуала правь CSS-блок `<style>`.
- Для интеграции формы сделай серверный обработчик вместо `alert`.
